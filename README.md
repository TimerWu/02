**需求分析文字描述**   
(1)**登入功能**:使用者可以登入及註冊帳號   
(2)**記事本**:使用者可以在這裡記錄重要的事項   
(3)**記帳功能**:使用者可以記錄每天的收入及支出，達到有效的金錢控管  
(4)**計算機**使用者可以使用簡單易懂的介面，做出基本的運算
